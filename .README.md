# RAG Chatbot with DeepEval

## Overview

This project implements a Retrieval-Augmented Generation (RAG) chatbot using:

- LangChain
- Ollama (LLaMA3)
- ChromaDB
- DeepEval

## Pipeline

PDF → Chunking → Embeddings → Vector DB → Retrieval → LLM → Response → Evaluation

## Evaluation

- Answer Relevancy
- Faithfulness

## Run

```bash
python3 app.py
```
